# 🌌 Luminary – AI Virtual Classroom

Luminary is an **AI-powered virtual classroom** that brings learning into an immersive environment using **3D avatars, voice narration, and real-time animations**.

It transforms any topic into an interactive lesson with a **virtual teacher in your space**, making learning visual, engaging, and accessible.

---

## 🚀 Features

- 🧠 AI-generated lessons using LLMs
- 🎙️ Real-time voice narration (ElevenLabs)
- 🎬 Visual explanations using Manim animations
- 🧑‍🏫 3D teacher avatar (USDZ + Mixamo)
- 🌐 Runs in browser + Apple Vision Pro (WebSpatial)
- ⚡ Fully interactive spatial learning experience

---

## 🛠️ Tech Stack

### Frontend
- React + Vite + TypeScript
- Tailwind CSS
- WebSpatial SDK
- Three.js (for 3D rendering)

### Backend
- Flask (Python)
- Gemini API (lesson generation)
- ElevenLabs API (voice synthesis)
- Manim (animation engine)

---

## ⚙️ How It Works

1. User enters a topic  
2. Gemini generates structured lesson  
3. ElevenLabs converts lesson → voice  
4. Manim generates animations  
5. 3D teacher presents content in real-time
